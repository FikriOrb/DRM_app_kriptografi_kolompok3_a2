# Android DRM WebView Wrapper

This folder contains the native wrapper source needed for TASK 4. Open this folder directly in Android Studio:

```text
C:\Projek kripto\android_wrapper
```

Required Gradle dependencies:

```gradle
implementation "androidx.appcompat:appcompat:1.7.0"
implementation "androidx.security:security-crypto:1.1.0-alpha06"
```

Set the WebView URL in `AndroidManifest.xml` metadata key `react_app_url`.
Use `http://10.0.2.2/...` for an Android emulator that talks to XAMPP on the host PC.
Use your PC LAN IP, such as `http://192.168.1.5:3000/`, for a physical Android device on the same WiFi.

The project currently does not include `gradlew.bat`, so command-line builds require Gradle in PATH or a wrapper generated by Android Studio. Android Studio can still sync and run the project from its own Gradle integration.

Release builds should keep WebView debugging disabled and should use HTTPS outside local development.
